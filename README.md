# Simple lil todo list

lets you keep track of tasks to do and keep track of previously completed tasks

Commands:
just typing a task allows that tast to be added to the active task list
-c/--complete [id for task to be marked as complete] allows you to mark a task as complete
-l/--list shows the current list of active 
-d/--delete [id for task to be deleted] allows you to delete a task from the active task list

-u/--update lets you update the current list removing all completed tasks and moving them to a seprate list
-lc/--listcomplete shows the list of completed tasks

-r/--reset lets you either reset all tasks, or just one of the two lists
